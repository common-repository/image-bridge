== Description ==
Image Bridge is a Light weight WordPress plugin that allows you to add custom image URLs, alt and title attributes to the featured images of your pages and posts and products.

== Installation ==
1. Upload the 'image-bridge' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the pages or posts or products edit page ,in the admin sidebar you see the 'image-bridge' meta box for adding custom image URLs, alt and title attributes to your featured images.

== Usage ==
1. Go to the post or page editor in WordPress.
2. In the meta box titled 'Image Bridge', enter the custom image URL, alt and title attributes for the featured image.
3. Save or update the post or page.
4. The featured image will now display the custom image with the specified alt and title attributes.

== Frequently Asked Questions ==
= How do I access the Image Bridge meta box? =
Go to the post or page editor in WordPress and look for the 'Image Bridge' menu item in the admin sidebar.

= Can I add custom image URLs, alt and title attributes to existing featured images? =
Yes, you can edit the post or page containing the featured image and update the 'Image Bridge' meta box with the new information.
